⚙️ Auto-PiP Settings

Configure which sites automatically trigger Picture-in-Picture

Enter one URL pattern per line. You can use:
• * to match any characters
• https://www.youtube.com/* matches all YouTube pages
• *://example.com/* matches http and https

📌 Common Examples:

  • https://www.youtube.com/*
  • https://www.netflix.com/*
  • https://vimeo.com/*
  • https://www.twitch.tv/*
